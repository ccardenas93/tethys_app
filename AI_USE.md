# AI Use Statement

This prototype was scaffolded with assistance from OpenAI Codex. The runtime app
does not call an AI model. Forecast values are generated from open meteorological
data, station observations, deterministic bias-correction logic, and verification
metrics.

Before contest submission, review the Tethys Geoscience Foundation AI Use Policy
and update this statement to match the final development process and any
additional tools used by the team.
